## StayChat Task

# Tech Stack
Python Flask
MongoDB Atlas
React (Vite)
Tailwind CSS

# Features
Add Item
Edit Item
Delete Item
Text Summarization (since the real one is paid so for that reason i didnt use that and instead of that i used a little mock ttype of that)

## How To Run

# Backend
cd Backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py

# Frontend
cd frontend
npm install
npm run dev
